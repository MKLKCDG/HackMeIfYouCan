## Installation

### Manuel Installation

1. Clone the repo
   ```sh
    git clone https://github.com/MKLKCDG/HackMeIfYouCan
   ```
2. Build docker image
   ```sh
    docker build -t MKLKCDG/HackMeIfYouCan .
   ```
3. Run container
   ```sh
    docker run 
   ```
4. Go to http://localhost:6619
